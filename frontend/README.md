# Admin Frontend

Simple React + Vite admin UI for the Logging & Audit module.

## Setup

```bash
cp .env.example .env
npm install
```

## Run (development)

```bash
npm run dev
```

Open http://localhost:5173

Ensure the backend is running on `VITE_API_URL` (default `http://localhost:8080`).

## Demo login

- Username: `admin`
- Password: `12345678` (unless overridden in backend `.env`)

## Build

```bash
npm run build
npm run preview
```
